# 💤 LazyVim based configuration

If you want to start clean you can do:

```
brew install neovim
git clone https://github.com/petros/nvim ~/.config/nvim
```

and if you want to make it yours right afterwards you can do:

```
rm -rf ~/.config/nvim/.git
```

and then initialize you own repository.

This is based on the starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.
